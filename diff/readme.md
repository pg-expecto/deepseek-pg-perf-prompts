# Библиотека промптов для анализа разностных отчётов pgpro_pwr

В этой папке собраны промпты, предназначенные для сравнительного анализа разностных отчётов `pgpro_pwr` между двумя интервалами снимков (`X1-X2` и `Y1-Y2`). Промпты помогают извлекать и интерпретировать статистику разделов **Load distribution** и **Top SQL**, формулировать гипотезы о ключевых факторах, вызвавших изменения, и определять способы их подтверждения или опровержения.

Все ответы должны оформляться в стиле `Philosophical_instruction_BETA_v5.1.md`, при этом цветовые метки необходимо заменять соответствующими текстовыми сокращениями:

- `"Подтверждено"`
- `"Вероятно"`
- `"Предположение"`
- `"Неизвестно"`

## Файлы промптов

| Файл | Назначение |
|------|---------|
| `prompt-diff-top-sql-analyze.txt` | Дополняет существующий аналитический отчёт (`analyze.txt`) выводами из `1-top-sql-1.txt`, уточняет или расширяет гипотезу о влиянии ключевого фактора на показатели **Load distribution**, приводит цепочку рассуждений и добавляет способы проверки/опровержения. |
| `prompt-diff-load_distribution.txt` | Извлекает JSON-массив `load_dist` из `pgpro_pwr_diff.X1-X2-Y1-Y2.clear.html` и строит сравнительную таблицу абсолютных значений метрик в формате Markdown для баз данных с признаком `db_filter: true`. Сопоставление полей соответствует `ТаблицаG.25.txt`. |
| `prompt-diff-load_distribution-analyze-1.txt` | Готовит аналитический отчёт, выделяющий ключевые отличия в показателях **Load distribution** между двумя интервалами, на основе ранее сформированной сравнительной таблицы (`1-load_distribution_table.txt`). |
| `prompt-diff-load_distribution-analyze-2.txt` | Формулирует гипотезу о влиянии ключевого фактора на метрики **Load distribution**, описывает цепочку рассуждений и предлагает способы подтверждения и опровержения гипотезы. |
| `prompt-diff-top-sql.txt` | Проводит сравнительный анализ статистики выполнения и ожиданий SQL-запросов из раздела **TOP-SQL** разностного отчёта, формулирует соответствующую гипотезу и предлагает методы её проверки. |

## Справочные описания таблиц

В папке также содержатся справочные таблицы, описывающие структуру и вычисление метрик, используемых при анализе:

- **`ТаблицаG.25.txt`** – Метрики распределения нагрузки (Total time, Executed count, I/O time, Blocks fetched, WAL и др.) и соответствующие им поля JSON.
- **`ТаблицаG.33.txt`** – Столбцы раздела Top SQL по времени выполнения (Query ID, Plan ID, JIT Time, времена CPU, число строк и т. д.).
- **`ТаблицаG.36.txt`** – Столбцы раздела Top SQL по времени ожидания ввода-вывода (времена чтения/записи, чтения/записи разделяемых, локальных и временных блоков и т. д.).

## Использование

1. Убедитесь, что необходимые входные файлы доступны:
   - Разностный отчёт `pgpro_pwr_diff.*.clear.html`.
   - Все предварительно извлечённые промежуточные файлы (например, `analyze.txt`, `1-top-sql-1.txt`, `1-load_distribution_table.txt`).
   - Соответствующие файлы описаний `ТаблицаG.*.txt`.
2. Передайте промпт и его входные файлы языковой модели.
3. Результатом обычно является структурированный аналитический отчёт (в основном используются списки, а не таблицы, согласно указанию: *"Даже если таблицы нагляднее - используй списки"*).
4. Все цветовые оценки должны быть преобразованы в перечисленные выше текстовые метки.

## Примечания

- Промпты предполагают знание расширений `pgpro_pwr` и `pgpro_stats` от Postgres Pro.
- Имена интервалов (`X1-X2`, `Y1-Y2`) являются плейсхолдерами и при реальном использовании должны заменяться фактическими идентификаторами отчётов.
