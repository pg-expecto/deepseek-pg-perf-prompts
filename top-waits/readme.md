### **Получение таблицы Top Waits**

Данная инструкция описывает процесс формирования исходной таблицы по ожиданиям СУБД по разделу "Top wait events" для дальнейшего использования в аналитических отчетах

#### **Необходимые файлы**
*   **Инструкция для нейросети**: `_prompt-instruction.txt` (доступна в [репозитории GitHub](https://github.com/pg-expecto/deepseek-pg-perf-prompts/blob/main/_prompt-instruction.txt))
>   Загрузите этот файл и сохраните локально.
*   **Шаблон для извлечения параметров**: `ТаблицаG.24.txt`
*   **Отчет `pgpro_pwr` в формате HTML**: `pgpro_pwr.X1-X2.clear.html`

---

#### **Формирование таблицы**
1.  Откройте **новый чат** DeepSeek.
2.  Загрузите в чат файл **`_prompt-instruction.txt`** (локальная копия).
3.  Следуя инструкции, отредактируйте файл `prompt-top-waits.txt`:
    *   Замените плейсхолдер `X1-X2` на фактические идентификаторы снимков (например, `01-02` или `A-B`) из имени первого отчета `pgpro_pwr`.
4.  Загрузите в чат **отчет** `pgpro_pwr` (`pgpro_pwr.X1-X2.clear.html`) и файл **`Таблица.G.24.txt`**.
5.  Отправьте исправленный файл `prompt-correspondence_table.source.1.txt` как запрос.
6.  Сохраните полученный результат в файл `top-waits.txt`.
